TransferLiquid TransferLiquid_Parameters .sila2.org.silastandard.CommandConfirmation

Dispense a predefined volume from a predefined source to a predefined destination on the target plate in x-y coordinates. This command provides more low-level access to the dispensing capabilities of the device than the Dispense Protocol Command. IMPORTANT! This command enables direct control of the axis movement and can thereby cause damage if used without thorough understanding of the device, and careful testing. We recommend to use DispenseProtocol for most users. Please refer to the SiLA 2 Automation Guide to learn more how to use this command.
