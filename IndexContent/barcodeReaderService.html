BarcodeReaderAvailability sila2.org.silastandard.Boolean

Query whether the barcode reader is available or not. This property reflects the state of the "UseBarcodeReader" property in the I.DOT Assay Studio configuration file. This property can be changed in the general settings of I.DOT Assay Studio and requires the barcode reader component to be installed in the device to be set to TRUE.

BarcodeReaderService

Service to read a barcode from a SBS format plate that has been placed in either the source tray or the target tray.

Reading barcodes with this feature is possible only if the I.DOT is equipped with a barcode reader. This can be checked via the property Barcode Reader Availability.

To sucessfully read a barcode, it must be positioned correctly on the plate, and the plate must be oriented in the tray so that the barcode is facing the barcode reader.
